# INRU Browser

Краткое описание проекта: современный, быстрый и безопасный веб-браузер.

## Описание

**INRU Browser** — это современный веб-браузер, ориентированный на скорость, безопасность и удобство пользователя. Проект включает в себя настройки для автоматизации процессов разработки и развертывания с помощью GitHub Actions.

## Структура проекта

```
.
├── .github/
│   └── workflows/
│       └── blank.yml    # CI/CD конфигурация
└── README.md            # Этот файл
```

## Использование

### Запуск CI/CD

Workflow автоматически запускается при:
- Пуше в ветку `main`
- Создании pull request в ветку `main`

Также можно запустить вручную через вкладку Actions на GitHub.
